# Image Interpolation

## 1. PPMBilinearInterpolation

PPMBilinearInterpolation performs bilinear interpolation for PPM file. 

## 1.1 Intended PPM format

```text
P3 [width] [height] 255
... data ...
... data ...
```

## 1.2 How to run

1. Run `ghci PPMBilinearInterpolation.hs`
2. Execute **main** function
